🎵 Volume Control by Hand Gesture
Control your computer's system volume using hand gestures in real time with the help of Computer Vision and Artificial Intelligence.

📌 Project Overview
This project uses a webcam to detect hand landmarks and changes the system volume based on the distance between the thumb and index finger.
The project is built using Python, OpenCV, MediaPipe, and Pycaw.

🚀 Features
- ✋ Real-time hand tracking
- 🔊 Control system volume with hand gestures
- 📷 Webcam-based interaction
- ⚡ Smooth and responsive performance
- 🤖 AI-powered hand landmark detection

🛠 Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy
- Pycaw
- Comtypes

Run the project
python volume_control.py

## 📷 Working
1. Open the application.
2. Allow webcam access.
3. Show your hand in front of the camera.
4. Move your thumb and index finger.
5. The distance between the fingers controls the system volume.

🎥 Output
The program detects the user's hand in real time using the webcam. By pinching (bringing closer) or separating the thumb and index finger,
the system volume is adjusted automatically.

* 🤏 Pinch fingers closer → 🔉 Volume Down
* ✋ Move fingers apart → 🔊 Volume Up
* 📊 Live volume percentage is displayed on the screen.
* ⚡ Smooth and responsive gesture recognition with real-time hand tracking.
